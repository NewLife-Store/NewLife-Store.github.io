告别 1.0 年代，迎接 2.0 时代
Created 2020-01-02| Updated 2020-01-02
Post View:

一路坎坷，一路荆棘。不算顺利的 2019 年，在一地鸡毛中，悄然的结束了。

这一年既有人生意义行的突破，也有自我否定的无奈。总的来说这一年算是成功参半，家庭与事业的博弈更加凸显出矛盾点。

在这一年中，经历了整个职业的转变，从用户方向，转换到现在的商家方向。从会员营销/增长，走向当前的商家赋能，商品活动道路。

1.0 的总结

整体总结：

关键词：折腾。

生活的事情在折腾，工作的事情在折腾。
可能是因为有了自己的家庭，曾经玩笑间不屑一顾的小事，开始在自己的生活中浮现，彼此之间在不段的适应着角色的转变。
变化的工作环境，打乱了自己既定的工作计划。但是本着学习即是成长，经历就会有收获。所以在业务与领导，内部与外方之间不断挣扎。

** 整体评价：一般（有满意有失望) **

一）家庭

家庭，算是这一年中最大的收获。

都说结婚是人生中最重要的几件事情之一。一番波折下来，也是身心俱疲。（不只是肉体的，还有精神的）

婚纱照：海南-蜈支洲岛（旅拍）
戒指：日系小众定制对戒
婚礼：十八线小县城最好的酒店
婚庆：十八线小县城比较知名的婚庆（专业性一般）

一天的婚礼，现场还是比较满意的。简单，清新，又不失格调的森系风格。

二）生活：

1：没有做好早睡早起，实在是太难了，但是已经意识到需要去改变了
2：对于健康可能意识到是一个更好的投资，已经在健身（和工作有些冲突，但是也会想办法解决），也要努力做好第一点的不足
3：对一些不好的习惯还是要有一定的克制。想到做的事情一定要及时去做。
有一些是沉迷于一件事情，无法自拔。比如说玩游戏停不下来，比如看电视时停不下来
（在工作中也有体现），有的难以理解的行为。

三）工作：

1：工作上这一年心理变化比较大，工作5年的发展历程是，学生-工作-社会，现在才对社会有了一些体悟（这一年就没白干）。
2：这一年在工作中虽然有一些突破，但是是有一些消极的，在后半年有一些改善，想通不要抱怨社会，要融入社会，社会,社会。

四）学习：

1：其实前半年是比较荒废的，但是发觉自己还是有很多金钱是好的欲望的，那只能投资自己了。
其实就是想明白立即要做。空想是不行的，什么事情要赚钱都是要努力的，都是非常辛苦的，对于不善交际的自己来说学习可能低成本，高效率了，还有一丝丝希望。
2：适合自己的学习方法，费曼技巧，虽然效率低，但是比不学习强点，还是有点效果的，而且刻意锻炼自己的意志力–还差好多，时刻提醒自己要精致。
书籍:《剑指offer》
MySQL: 函数式接口和stream
3: 英语-假装当做一个学习的兴趣吧，也是为了辅助编程。

2.0 时代：

1：希望意志力不需要刻意锻炼，成为习惯吧。
2：英语不求效果，每天都在学习就行。
3：体重稳定到140斤。
4：每天能早睡1分钟（就是一分钟，下意识的就躺被窝就睡觉）。
5：完成函数式接口和stream视频录制。
6：考研-考研-考研-考研。
7：多学会沉默，沉默是金，耐心和不要皱眉。不要抢话，不要自以为是。
